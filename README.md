# Music-Queue
Music-Queue is a dynamic music management tool that lets you search, organize, and navigate your favorite songs efficiently

## Features
- Search & Add Songs: Search for songs via YTMusic and see the top 5 results. Add any selection directly to the queue at a specified position.
- Queue Navigation: Move forward and backward through the queue seamlessly.
- Efficient Queue Management: The queue is implemented as a doubly linked list, allowing constant-time addition and removal (O(1)).

## Technology Stack
- Queue Implementation: Doubly linked list (fast insertions/removals)
- Search Engine: YTMusic API
